## Decisions Made

- **Rollback is admin-only**: Only `ori@compulocks.com` (or `ADMIN_EMAILS` env var) can see history and restore. Standard users cannot.
- **Version semantics**: 0.x = draft, 1.x = published. Matches standard draft/release nomenics per user spec.
- **Paint "Clear" overrules** all other paint options — selecting it clears colors and no color input is shown.
- **FeatureSelector cleaned**: Removed `customization` and `dispatch` props entirely; it now only manages features. Customization lives in Section 01.
- **Snapshot on every PATCH**: History is created server-side on each save, not client-side. Client just sends content; server handles the snapshot. Max 20 entries prevents unbounded growth.
- **page.tsx → server component**: Needed to resolve `isAdmin` from NextAuth session on server without a round-trip fetch. Thin wrapper pattern: server reads session, passes `isAdmin` boolean to client component.
